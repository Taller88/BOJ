####  분할 정복(Divide and Conquer)
     * 색종이 만들기
       - 접근방법 
       1. 전체 n이 다 같은 숫자인지 check 
       2. n을 반으로 나눈 상태에서 check
       3. n을 나눈 상태에서 x값에 +n/2
       4. n을 나눈 상태에서 y값에 +n/2
       5. n을 나눈 상태에서 x값에 +n/2, y값에 +n/2
       6. 재귀 -> 4분할
       
     * 종이의 개수
       - 접근방법 
       1. 전체 n이 다 같은 숫자인지 check 
       2. n을 3으로 나눈 상태에서 check
       3. n을 나눈 상태에서 x값에 +n/3
       4. n을 나눈 상태에서 y값에 +n/3
       5. n을 나눈 상태에서 x값에 +n/3+n/3, y값에 +n/3
       6. n을 나눈 상태에서 x값에 +n/3, y값에 +n/3+n/3
       7. n을 나눈 상태에서 x값에 +n/3+n/3
       8. n을 나눈 상태에서 y값에 +n/3+n/3
       9. n을 나눈 상태에서 x값에 +n/3+n/3, y값에 +n/3+n/3
       9. n을 나눈 상태에서 x값에 +n/3, y값에 +n/3

       10. 재귀 -> 9분할
